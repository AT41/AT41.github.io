Password Strength Measurer

This is a password strength measurer my group created for a Computer Security course. I would have make this using NodeJS since I was confident with creating a nice UI with HTML and CSS, but since my group members weren't very familiar with JavaScript nor TypeScript, we decided to work with Java. I had learnt how to use Java Applets a couple of years ago, so I did all of the UI, as well as a couple of the password guessing algorithms.
